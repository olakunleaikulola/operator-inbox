# Operator Inbox

A lightweight workflow inbox for marketers, product builders, and solo operators to capture ideas, priorities, and active work in one place.

## Goal

Build a small, useful tool that helps turn scattered work inputs into one clear next move.

## V1

The first version will let a user:

- add an item
- assign a category
- assign a status
- view current items
- filter items
- get an AI-assisted recommended next move
